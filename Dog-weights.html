Data cleaning

At initial stage, all the variables are stored as characters, which might not be what we desired. In order to produce tidy dataset, we need to:

  • Convert Date variable to the right data class by as.Date. We can also get the number of days easily through this function.

  • We want to show each Dogs’ weight data in pounds. Therefore, separate function is applied to divide Lbs and Oz. After that, we’re allowed to convert the weights by \(1Lbs = 16Oz\).

Here is the head and tail of the clean data: (Left: Head, Right: Tail)

dogweights = read_excel("./data/dog _weights.xlsx", range = "B3:E185")[-77,] %>% 
  janitor::clean_names() %>% 
  mutate(
    date = as.Date(as.numeric(date), origin = "1899-12-30")) %>%
  # raisin
  separate(raisin, into = c("raisin_lbs", "raisin_os"), sep = " ") %>% 
  mutate(raisin_lbs = round(as.numeric(raisin_lbs) + as.numeric(raisin_os)/16, digits = 2)) %>%
  # simone
  separate(simone, into = c("simone_lbs", "simone_os"), sep = " ") %>%
  mutate(simone_lbs = round(as.numeric(simone_lbs) + as.numeric(simone_os)/16, digits = 2)) %>%
  # gagne
  separate(gagne, into = c("gagne_lbs", "gagne_os"), sep = " ") %>%
  mutate(gagne_lbs = round(as.numeric(gagne_lbs) + as.numeric(gagne_os)/16, digits = 2)) %>% 
  select(date, raisin_lbs, simone_lbs, gagne_lbs)

Following up, time for some data visualizations.

Visualization

Before we get started, weight data is spread across three columns, which correspond to three different puppies. To facilitate further process, we apply pivot_longer and change the data to long format.

weights_time = 
  dogweights %>%
  rename(
      Raisin = raisin_lbs, 
      Simone = simone_lbs, 
      Gagne = gagne_lbs) %>%
  pivot_longer(
    Raisin:Gagne,
    names_to = "Name",
    values_to = "Weights")

Weights of dogs

We can get detailed information by placing cursor over the graph. (Mean, Median, Max/Min) There’s an outlier (7.94Lbs) in Raisin’s weight that doesn’t seem quite realistic, so we think of it as a typo and will remove it in following analysis. We can infer from the plot that Gagne and Simone might belong to a small-sized family like Pomeranians, while Rasian should belong to another larger-sized breed.

plot_weights = 
  ggplot(weights_time, aes(x = Name, y = Weights, fill = Name)) +
  labs(title = "Weights of dogs",
       x = "Dog",
       y = "Lbs") + 
  stat_summary(fun.y = median, geom = "point", color = "blue", size = 3) + 
  geom_boxplot(alpha = 0.5, size = 0.5, color = "blue")

ggplotly(plot_weights)

Weights over time

The light green shaded areas indicate the time when these dogs are not under their owner’s care. By zooming in these areas, we can find that the dogs’ weights continue to drop until January 2019 because their owner was constantly boarding them with Weber or Kennel. After that, the dogs’ weights rose slowly and steadily while their owners stayed with them. The Light blue dashed lines indicate dates that the dogs go on a haircut. In the enlarged plot, we can find that Raisin lost more weight after haircut than Gagne and Simone did. Therefore, we assume that Raisin is a member of the long-haired breed, while the others are not.
